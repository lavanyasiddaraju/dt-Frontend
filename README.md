# dt-Frontend
Frontend implementation of the DTthon Technical Project Management interface using HTML, CSS, and vanilla JavaScript, built by closely following the provided Figma design and dynamic JSON data.

Deep Thought – Technical Project Management
DTthon Frontend Challenge 

This repository contains a frontend implementation of the Technical Project Management interface based on the provided Figma design and JSON data as part of the DTthon Frontend Challenge.
The goal was to build a clean, structured UI and render assets dynamically using HTML, CSS, and Vanilla JavaScript.

Task 1 – Webpage Creation
Implemented the layout using HTML & CSS as per the Figma design.
Built reusable asset containers and expandable sections.
Maintained visual consistency with the design.

Task 2 – Dynamic Functionality
Rendered project data dynamically from a local JSON file.
Avoided hardcoding of content.
Implemented reusable components using JavaScript loops.
Handled multiple asset types (video, input, article, external links).

Features are:
Two-column responsive card layout.
Reusable asset container component.
Expandable / collapsible sections.
Conditional rendering based on asset type.
Clean and scalable code structure.

Tech Stack Used:
HTML5
CSS3
Vanilla JavaScript

How to Run
1. Clone or download this repository.
2. Open index.html using Live Server or any local server.
3. Ensure all files are in the same directory.

 Notes:
This implementation is a simplified representation of the DeepThought platform, as per Task-1 and Task-2 scope.
Backend logic and data persistence are out of scope.
Some external links may show server-side errors (e.g., 502), which are not frontend issues.

Finally,This project focuses on clarity, structure, and best frontend practices while strictly adhering to the challenge guidelines.
